# RegPilot

Agentic RAG chatbot that classifies an AI system against the EU AI Act, retrieves the
applicable Articles, computes compliance deadlines and emits a roadmap with citations.

Work in progress — see `pyproject.toml` for dependencies. Full docs land with the
final commit; for now `docker compose up --build` boots Ollama + the Streamlit UI on
`http://localhost:8501`.
